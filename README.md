# Mathematical Logic & Algorithmic Challenges in Python

## 📌 Overview
This repository is dedicated to solving complex mathematical logic, discrete mathematics, and algorithmic challenges using pure Python. The goal is to sharpen problem-solving skills, optimize algorithmic efficiency (Time & Space Complexity), and maintain a consistent coding habit.

## 🛠️ Environment & Mobile Workflow
As part of my commitment to adaptability and resource optimization, this entire repository is developed, tested, and managed directly from a mobile Linux environment:
* **Engine:** Pydroid 3 (Local Python execution environment)
* **Terminal:** Termux (Linux emulation & Git version control)

## 📂 Repository Structure
The challenges are organized by mathematical and logical categories:
* `01_propositional_logic/`: Truth tables, logical gates simulators, and boolean algebra simplifiers.
* `02_number_theory/`: Prime factorization, modular arithmetic, and Euclidean algorithms.
* `03_combinatorics_graphs/`: Permutations, combinations, and graph traversal algorithms (BFS/DFS).
* `04_dynamic_programming/`: Optimization problems and memoization patterns.
